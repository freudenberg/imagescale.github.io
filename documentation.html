Managing Automations

Sign in

Once you have logged in with your username and password, you are on the automation screen.

Create new automation

Click New Automation. A new screen appears with various options. Let's set up an automation to scale and crop your images.

First have a look at the two tables. The left hand table under the headline Actions allows you to compose your automation. The right hand table under the headline Palette contains the available actions.

  1. Enter a label, e.g. "small-crop"
  2. In the Actions table find the row with the Crop action and remove it by clicking the button in the rightmost column of that row.
  3. Find the ScaleAndCrop action in the Palette table, click on the word ScaleAndCrop and drag it over the Actions table. Make sure to drop the action between the Load and Render actions. If it does not end up where it ought to be drag it around a bit.
  4. ScaleAndCrop has some parameters: width, height, yOffset and xOffset. Enter some values for xOffset and yOffset, e.g. 20 and 50.
  5. Click the Save button.

Modify existing automation

Nowadays some images taken by smartphones store camera rotation data and if this is not taken into account the image might show up with an unintended rotation.

ImageScale has an action for correcting this automatically: AutoRotate. You can add as many actions to an automation as you want. Beware, sometimes the order matters. All actions are executed in the order you place them in the table from top to bottom.

  1. Drag the AutoRotate action from the Palette table to the Actions table. Make sure it is between Load and Render.
  2. Click the Save button.
